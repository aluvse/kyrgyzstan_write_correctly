# Кыргызстан: Пишем правильно

Автоматическое расширение для браузера, которое помогает соблюдать нормы официального написания названия нашей страны в интернете.

### Описание

Расширение в реальном времени находит устаревшие или ошибочные формы написания названия страны (такие как "Киргизия") на веб-страницах и заменяет их на официальную версию: **Кыргызская Республика (Кыргызстан)**.

### Основные функции

- **Автоматическая замена**: Работает незаметно для пользователя при загрузке страниц.
- **Статистика**: Ведет подсчет исправленных слов, который отображается в интерфейсе расширения.
- **Гибкие настройки**:
- Можно временно отключить замену через переключатель в настройках.
- Можно добавить текущий сайт в «белый список» (отключить замену только для конкретного домена).

- **Конституционная основа**: Напоминает об официальном названии согласно статье 1 Конституции КР.

### Как это работает

Расширение анализирует текстовые узлы страницы и при обнаружении ключевых слов заменяет их на правильные, подсвечивая изменение. При наведении на исправленное слово отображается краткая справка.

### Техническая информация

- **Технология**: WebExtensions API (Manifest V3).
- **Поддержка**: Работает в Firefox и браузерах на базе Chromium (Chrome, Edge).
